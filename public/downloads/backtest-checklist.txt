백테스트 체크리스트 - 메린이(META Child)
==========================================

테스트 전 확인사항 10가지

1. 데이터 품질 확인
   - 틱 데이터 vs 분봉 데이터 차이 이해
   - 데이터 소스의 신뢰성 검증

2. 스프레드 설정
   - 고정 스프레드 vs 변동 스프레드
   - 실제 거래 시간대별 스프레드 반영

3. 슬리피지 시뮬레이션
   - 시장가 주문 시 예상 슬리피지
   - 뉴스 시간대 슬리피지 고려

4. 수수료/스왑 반영
   - 브로커별 수수료 구조 확인
   - 오버나이트 스왑 비용 계산

5. 시간대 설정
   - 서버 시간 vs 로컬 시간
   - 여름/겨울 시간 변경 고려

6. 최적화 과적합 방지
   - 인샘플/아웃샘플 분리
   - 워크포워드 분석 적용

7. 드로우다운 분석
   - 최대 드로우다운 허용 범위 설정
   - 연속 손실 시나리오 검토

8. 포지션 사이징
   - 고정 랏 vs 비율 기반
   - 마진 요구사항 확인

9. 시장 조건 변화
   - 트렌드/레인지 구간 별도 분석
   - 변동성 변화 시 성과 비교

10. 실전 적용 계획
    - 데모 테스트 기간 설정
    - 라이브 전환 기준 명확화
